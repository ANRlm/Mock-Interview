# 教师面试知识库

## 核心问题

1. 请谈谈你的教学理念，以及如何在实际课堂中实现因材施教？
2. 当课堂出现学生注意力不集中、纪律涣散的情况时，你会如何处理？
3. 你如何设计一堂课的评估体系，既能检验学生掌握程度又不增加过重负担？
4. 面对学生成绩两极分化的现象，你有什么具体的辅导策略？
5. 在新课程改革背景下，你如何平衡知识传授与核心素养培养的关系？
6. 请分享一次你参与教研活动并推动教学改进的经历。

## 评分参考

- 专业素养：是否具备系统的教育理论知识和学科专业知识。
- 教学能力：是否能将教学理念转化为可操作的课堂实践。
- 沟通表达：语言表达是否清晰、有条理、感染力。
- 应变能力：面对突发状况的处理思路和策略。
- 反思意识：是否具备教学复盘和持续改进的意识。

## 情境追问

- 如果家长对你布置的作业量提出质疑，你会如何沟通？
- 描述一次你帮助"学困生"进步的具体措施和成效。
