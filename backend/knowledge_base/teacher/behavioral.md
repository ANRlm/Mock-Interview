# 教师面试行为问题（STAR格式）

## 情境类问题

### 问题一：课堂管理

**请描述一次你成功处理课堂突发纪律问题的经历。**

- **情境（S）**：当时班级的整体氛围、学生人数、课程背景。
- **任务（T）**：你作为教师需要达成的目标。
- **行动（A）**：你具体采取了哪些措施（教学调整、管理手段等）。
- **结果（R）**：最终的处理效果和后续影响。

### 问题二：家校沟通

**请分享一次你需要与家长进行深度沟通的经历，当时面临什么挑战？**

- **情境（S）**：学生的问题表现、家长的态度和期望。
- **任务（T）**：你需要达成的沟通目标。
- **行动（A）**：你做了哪些准备工作、采用了什么沟通策略。
- **结果（R）**：沟通的最终效果和后续跟进情况。

### 问题三：教学改进

**描述一次你基于学生反馈或成绩数据主动改进教学的经历。**

- **情境（S）**：你发现了什么问题或收到了什么反馈。
- **任务（T）**：你需要解决的教学痛点。
- **行动（A）**：你进行了哪些教学调整或尝试。
- **结果（R）**：改进前后的对比数据和自我反思。

### 问题四：跨学科协作

**请谈谈你参与教研组或跨学科协作的经历，你扮演了什么角色？**

- **情境（S）**：协作的背景、参与成员、目标设定。
- **任务（T）**：你在团队中的职责和分工。
- **行动（A）**：你提出了什么建议或承担了什么工作。
- **结果（R）**：协作的成果和你的收获。

## 评分参考

- 故事完整性：STAR四要素是否完整、逻辑是否连贯。
- 真实性：案例是否具体、可信，有细节支撑。
- 反思深度：是否体现自我反思和经验总结能力。
- 能力迁移：是否从中提炼出可复用的教学策略。
