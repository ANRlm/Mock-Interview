# 医生面试知识库（Phase 1）

## 核心问题

1. 请分享一次让你印象深刻的临床诊断经历，当时你是如何逐步明确诊断思路的？
2. 面对疑难病例时，你会如何平衡循证医学与个人临床经验的关系？
3. 你如何向患者解释复杂的病情和治疗方案，确保患者真正理解并配合治疗？
4. 在急诊抢救中，你如何快速判断病情的轻重缓急并合理分配医疗资源？
5. 谈谈你对医患关系的理解，以及你在实际工作中如何维护良好的医患沟通。

## 评分参考

- 临床思维：是否能展现系统的诊断逻辑和鉴别诊断能力。
- 沟通能力：是否能用通俗语言解释专业术语，体现患者为中心的态度。
- 应变能力：在高压环境下是否能保持冷静、做出合理决策。
- 职业素养：是否体现医学伦理、循证理念和持续学习的意识。
- 团队协作：是否重视多学科协作和院内转诊流程。

## STAR 情境追问

- 描述一次你发现并纠正上级医师诊疗方案潜在问题的经历。
- 描述一次患者家属对治疗方案产生重大误解时的沟通处理过程。
