# 律师情景面试题库

## 复杂法律情景题

### 情景一：跨境商业纠纷

**背景**：某中国制造商与一家美国品牌商签订独家代理协议，协议约定适用美国加州法律并约定在洛杉矶仲裁。合作三年后，美方突然以"违反排他性条款"为由发出终止函，并冻结了中方在美的应收账款约 2000 万美元。中方制造商认为美方是为摆脱独家代理束缚、另寻其他供应商而故意找借口。

**问题**：

1. 如果你代表中方制造商，你将如何着手分析这起纠纷？
2. 在洛杉矶仲裁和在中国法院起诉之间，你会如何为当事人选择更有利的救济途径？
3. 面对美方声称的"违反排他性条款"，你需要收集哪些证据来反驳？
4. 如何评估和应对仲裁裁决在两国的承认与执行风险？

**考察维度**：跨境纠纷策略、国际商事仲裁、证据链构建、风险预判

---

### 情景二：刑事辩护与证据审查

**背景**：你的当事人是一名企业高管，被指控在企业并购过程中向国家工作人员行贿。检察院已掌握部分转账记录和证人证言，但现有证据中存在几处关键矛盾：转账时间与并购关键节点不吻合；证人证言前后不一致；且有证据显示部分资金流向与当事人无关。

**问题**：

1. 作为辩护人，你将如何审查和运用这些证据矛盾？
2. 如何在不暴露当事人商业机密的前提下，提出合理的无罪辩护意见？
3. 面对检察院的举证压力，你会考虑申请调取哪些新证据？
4. 如果当事人坚持做无罪辩护，但你认为认罪认罚可能更有利，你会如何与当事人沟通？

**考察维度**：刑事辩护技巧、证据审查能力、刑民交叉思维、当事人沟通

---

### 情景三：劳动仲裁与群体性事件

**背景**：某互联网公司因业务转型决定大规模裁员，涉及员工 300 余人。公司 HR 在未充分协商的情况下，单方宣布裁员方案，并要求员工在 3 天内签署解除协议。部分员工情绪激动，在公司门口拉起横幅并在网上发布消息，引发媒体关注。

**问题**：

1. 作为公司法律顾问，你如何协助处理这一群体性事件？
2. 如何在维护公司利益的同时，避免激化矛盾导致更大舆论危机？
3. 对于拒绝签署协议的核心技术员工，你建议公司采取什么策略？
4. 如果员工集体申请劳动仲裁，公司应如何准备应对？

**考察维度**：劳动法实务、群体性事件处理、舆情应对、谈判博弈

## 评分参考

- **法律分析**：对相关法律规定的理解深度和适用准确性。
- **策略规划**：解决方案的系统性和前瞻性。
- **利益平衡**：在各方利益冲突中寻找平衡点的能力。
- **风险意识**：对潜在风险的认识和防范措施。
- **应变能力**：面对复杂局面的判断力和执行力。
