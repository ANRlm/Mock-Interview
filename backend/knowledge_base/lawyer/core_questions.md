# 律师面试知识库（Phase 1）

## 高频问题

1. 请介绍你最有代表性的诉讼案件，重点说明你在案件中担任的角色、争议焦点以及最终结果。
2. 当代理的案件面临不利证据时，你会如何制定反驳策略？
3. 如何在维护当事人利益与遵守律师职业道德之间取得平衡？
4. 请描述一次你成功说服法官采纳你法律观点的经历。
5. 在处理多起并行案件时，你如何合理分配时间和精力？
6. 你如何看待商事诉讼与非诉业务相结合的发展趋势？

## 评分参考

- 专业能力：是否对法律条文、程序规定有准确理解与灵活运用。
- 案件分析：是否能抓住争议核心，提出切实可行的代理方案。
- 沟通表达：逻辑是否清晰，能否用简洁语言阐述复杂法律问题。
- 职业素养：是否体现律师执业规范、保密义务和利益冲突意识。
- 应变能力：面对突发情况时的反应与处理策略。

## STAR 情境追问

- 描述一次你在庭审中遇到意外情况（如对方突袭证据）时的应对过程。
- 描述一次你为当事人提供法律方案时，需要在多个选项中权衡利弊的经历。
